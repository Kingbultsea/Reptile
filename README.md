很久之前帮别人弄的小爬虫 拿来爬知乎的
现在知乎接口有变动了 变得更加简单 所以来维护下这个项目
知乎应该是处于SEO问题而变简单的？页面也静态了 

很简单的利用链式promise  和superagent  就完成了
页面也很简单 单纯的vue 并没有用webpack 因为代码量并不大 

因为知乎是防盗的 需要用谷歌跨域来获取图片
不要直接访问3000  手动打开index.html吧(无自动弹出)
